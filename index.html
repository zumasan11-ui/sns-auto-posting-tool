広告の勝因分析

広告の勝因分析は、広告分析・ビジネスモデル分析の投稿素材を作成し、 ユーザー本人が管理するTikTokアカウントへ短尺動画を投稿するためのWebサービスです。 投稿はユーザー本人のTikTok認可に基づいて実行されます。

Official website: https://zumasan11-ui.github.io/sns-auto-posting-tool/

Service Functions

Content management

ユーザーが作成した広告分析文、投稿キャプション、短尺動画素材を管理します。

TikTok posting

ユーザー本人が認可したTikTokアカウントに、生成済みMP4を投稿します。

TikTok Integration

  • Login Kitでユーザー本人のTikTokアカウントを認証します。
  • user.info.basic は、投稿先アカウントを確認するために使用します。
  • video.publish は、Content Posting APIのDirect Postで本人アカウントへ投稿するために使用します。
  • Redirect URIは、この公式サイトと同じドメイン配下の /tiktok/callback/ です。

User Control

本サービスは、ユーザー本人が明示的に承認した投稿のみを扱います。 第三者アカウントへの投稿、ユーザーの許可なしの投稿、またはTikTokのポリシーに反する投稿は行いません。
